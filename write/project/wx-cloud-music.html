体验mpvue仿网易云音乐手机版

Younglina 2018/8/31 学习mpvue小程序

# my-project

在尝试开发了一个微信小程序之，后发现有 mpvue 这么个东西可以来开发小程序以后就忍不住了，所以把小程序和我用 vue 开发过的版本结合修改以后，再尝试用 mpvue+iview-weapp 开发一个版本。
后台接口用的是NeteaseCloudMusicApi (opens new window),感谢大佬的分享

git 地址 (opens new window)，demo 项目代码质量不是很好，哈哈哈哈多多见谅 😁

# 项目搭建

  • 最开始还是要把微信小程序开发工具 (opens new window)下载一下
  • mpvue 项目建立，跟着官网走一遍是最快的 快速上手 (opens new window)
  • 支持 iview-weapp，还是跟着官网快速上手 (opens new window)，这里就一个地方要注意，把下载下来的iview-weapp 项目的 dist 文件夹（可以改名字）是放到你项目编译后的dist 目录下，iview-weapp 官网没有效果的展示，所以建议扫它上面的二维码查看效果以后对照它的代码进行开发
  • 最后就是用微信小程序开发工具打开项目(不是 dist 目录)，用你常用的编译器编写代码（src 目录下）

# vuex 方面

还是和 vue 里面的使用一样，简单的
/src/store/store.js

import Vue from 'vue'
import Vuex from 'vuex';

Vue.use(Vuex);

export default new Vuex.Store({
  state: {
   oneState:'one'
    }
  },
  mutations: {
    setItem: (state,data) => {
      state.oneState = data
    },
  },
  getters : {
    getItem (state) {
        return state.oneState
    },
});
